【中文摘要】：
类不平衡问题是机器学习研究领域经常遇到的问题，会对分类结果产生一定的负面影响，目前有效的解决方案是对数据集中少数类别的数据进行扩充。针对图片数据集来说，现在主流的方法是通过过采样技术来增加样本的数量，这种方法虽然能缓解类不平衡问题，但是也会带来例如过拟合等方面的问题。生成对抗网络作为神经网络的一种，其主要用途在于制造与真实样本十分接近的人工样本。通过输入真实数据，网络模型经过生成模型和判别模型二者博弈的过程，输出的是和输入的数据非常类似的样本，此种算法可以有效缓解类不平衡问题带来的影响。 在生成对抗网络模型的基础上，通过在建模时对两个模型增加控制条件来输出指定类别的人造数据，可以专门针对数据集中的少数类数据进行扩充。使用这种条件生成对抗网络可以避免原始生成对抗网络模型生成数据的不确定性，对于类不平衡问题来说更有现实意义。主要工作如下： （1）了解不平衡数据集对分类问题的影响，对现有的数据扩充方法进行调研，熟悉各种方式的特点，并总结出其存在的问题。 （2）使用条件生成对抗网络模型来生成人工样本，既达到了数据扩充的目的，又避免了现有方法所存在的问题。 （3）研究相关图片数据集，根据实验要求选择出不平衡数据集，同时对平衡数据集进行处理来制造少数类，供算法研究的使用。 （4）通过训练不平衡数据集，生成少数类的人工样本，使用分类算法对扩充前后的数据集进行训练，发现分类精度有所提升，证明生成的数据具有一定的有效性。
【英文摘要】：
Class imbalance is a problem often encountered in the field of machine learning research, which will have a certain negative impact on the classification results. At present, the effective solution is to expand the data of a few categories in the data set. For the image data set, the current mainstream method is to increase the number of samples through oversampling technology. Although this method can alleviate the problem of class imbalance, it will also bring problems such as overfitting. As a kind of neural network, the main purpose of generating antagonistic network is to produce artificial samples very close to real samples. By inputting real data, the network model generates a game between the model and the discriminant model, and the output is a sample which is very similar to the input data. This algorithm can effectively alleviate the impact of the class imbalance problem. On the basis of generating the antagonistic network model, the artificial data of the specified category can be output by adding control conditions to the two models during the modeling process. Using this condition to generate the antagonistic network can avoid the uncertainty of the data generated by the original antagonistic network model. The main work of is as follows: (1) Understanding the impact of unbalanced data sets on classification problems, deeply understanding the characteristics of various methods, and summarizing the existing problems.by investigating the existing data expansion methods. (2) Using conditional generation antagonistic network model to generate artificial samples not only achieves the purpose of data expansion, but also avoids the problems existing in existing methods. (3) Studying related picture data set, select unbalanced data set according to the requirement, and process balanced data set to produce a few classes for algorithm research. (4) By training the unbalanced data set and generating a small number of artificial samples, the classification algorithm is used to train the data set before and after expansion, and the classification accuracy is found to be improved, which proves that the generated data has certain validity.
